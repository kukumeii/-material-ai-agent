# Material AI Agent with Xiaomi MiMo

> 基于小米MiMo大模型构建的工程材料性能智能预测与成分优化自动化Agent，面向含能多主元合金与化学链燃烧铁矿石氧载体体系。

---

## 🎯 项目背景与目标
传统材料研发依赖大量重复实验，仅一个体系的成分筛选与性能验证就需数月至半年时间，且难以对非线性关联的关键影响因素进行系统性分析。本项目旨在通过AI驱动的自动化数据处理、模型训练与可解释性分析，构建“数据-模型-优化-验证”的全流程闭环，解决传统研发中成本高、周期长、机理难解释的核心痛点。

---

## ⚙️ 系统架构与核心模块
本Agent采用模块化设计，支持多轮长链推理与多工具协同调用，核心逻辑流程如下：

### 1. 数据自动清洗与标准化模块
- 批量读取CSV/Excel/文献格式的实验数据
- 自动处理缺失值、异常点与单位不一致问题
- 生成统一格式的结构化数据集
- 自动完成数据分布与相关性的初步可视化分析

### 2. 物理引导特征工程与筛选模块
- 结合材料科学先验知识，自动生成30+物理描述符
  - 含能合金：原子半径、电负性、熔点/模量错配度、堆垛因子
  - 氧载体：Fe₂O₃含量、粒径、反应温度、氧燃比、煤化程度
- 通过遗传算法与皮尔逊相关性分析，自动筛选关键影响特征
- 有效解决小样本数据下的过拟合问题

### 3. 多模型对比训练与优化模块
- 自动生成并训练SVR、随机森林、XGBoost、ANN等模型代码
- 自动完成5折交叉验证与超参数贝叶斯优化
- 输出各模型性能对比报告（R²、MAE、RMSE）
- 快速锁定最优预测模型

### 4. 可解释性分析与关键因素识别模块
- 基于最优模型自动生成SHAP/LIME可解释性报告
- 量化各关键特征对目标性能的影响权重与作用方向
- 明确核心影响因素的作用机制，为机理分析提供可解释依据

### 5. 反向成分/工艺设计与验证模块
- 根据用户输入的目标性能指标，自动生成满足约束条件的成分/工艺优化方案
- 结合模型预测结果给出理论性能评估
- 生成实验验证建议，实现从“正向预测”到“反向设计”的闭环

## 📊 应用成果与性能表现
本Agent已在两项课题研究中投入实际使用：

### 1. 含能多主元合金拉伸屈服强度预测
- 数据集：88组高置信度铸态合金实验数据
- 最优模型：SVR，测试集R² = 0.928
- 关键发现：熔点错配度是影响屈服强度的第一核心因素
- 已辅助完成Ti-Zr-Nb-Ta体系的成分反向设计与实验验证

### 2. 铁矿石氧载体燃烧效率预测
- 数据集：101组煤/煤焦化学链燃烧实验数据
- 最优模型：GBDT，测试集R² = 0.931
- 关键发现：煤化程度(C/V)、氧燃比、Fe₂O₃含量为核心影响因素
- 为低成本高性能氧载体的工业筛选提供了数据支撑

### 3. 整体效率提升
- 日均调用MiMo接口约300–500万Token
- 将材料性能预测与成分优化的研发效率提升了70%以上
- 生成的数据分析报告与模型代码已直接用于两篇学术论文的撰写
